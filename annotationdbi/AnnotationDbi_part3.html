In this part I want to explore the chromosomes of the human genome. We have 23 chromose pairs, 22 autosomal pairs which contain essentially the same genes (albeit in slightly different versions, called alleles) and the gonosomes (or sex chromosomes) X and Y, plus the mitochondrial (MT) chromosome. X and Y contain different genes and in this capacity encode sex-related traits. The MT is the remaining genetic information from what initially started as a symbiosis. The majority of the mitochondrial genes were lost over time or have been incorporated into the main nuclear DNA, but a handful of genes remains to this day. MT genes are interesting because they are solely inherited via the female line as they come with the mitochondria in the fertilized egg.

I am again using the AnnotationDbi and EnsDb.Hsapiens.v79 packages. This time I want to extract the following information for each gene:

  • on which chromosome it lives
  • the length of the chromosome it lives on (in base pairs, bp)
  • whether it is on the plus or minus strand
  • and the coordinates of start and end position.

Based on start and end position of each gene I can calculate its length in base pairs (bp).

library(AnnotationDbi)
library(EnsDb.Hsapiens.v79)
edb <- EnsDb.Hsapiens.v79

# get all gene IDs (Ensembl gene ID)
keys <- keys(edb, keytype = "GENEID")

chromosomes <- AnnotationDbi::select(edb, keys = keys, columns = c("SEQLENGTH", "SEQNAME", "SEQSTRAND", "GENESEQSTART", "GENESEQEND"), keytype = "GENEID")

# convert chromosome and strand to factors
chromosomes$SEQNAME <- as.factor(chromosomes$SEQNAME)
chromosomes$SEQSTRAND <- as.factor(chromosomes$SEQSTRAND)

# calculate gene length
chromosomes$GENELENGTH <- chromosomes$GENESEQEND - chromosomes$GENESEQSTART

How many bases of each chromosome are taken up by genes?

In order to calculate the sum of gene lengths for each chromosome I am using the aggregate() function. We can already see that there are many more unique chromosome identifiers than the 25 regular chromosomes we had expected:

gene_bases <- aggregate(chromosomes$GENELENGTH, by = list(Chromosome = chromosomes$SEQNAME), FUN = sum)
length(unique(gene_bases$Chromosome))
## [1] 319

This is because some of the genes have not yet been successfully mapped to a chromosome. Genes with seqnames that start with âKIâ or âGLâ for example, currently live on scaffolds where the chromosomal location is not yet known.

How many genes are on each chromosome?

To count the number of genes per chromosome I am using the table() function.

chromosomes_num_table <- as.data.frame(table(chromosomes$SEQNAME))

chromosomes_num_table_other <- chromosomes_num_table[-which(chromosomes_num_table$Var1 %in% chrom), ]
scaffolds <- data.frame(Var1 = "S", Freq = sum(chromosomes_num_table_other[grep("^KI[0-9]+|^GL[0-9]+", chromosomes_num_table_other$Var1), "Freq"]))

chromosomes_num_table <- chromosomes_num_table[which(chromosomes_num_table$Var1 %in% chrom), ] %>%
  rbind(scaffolds)


How long are the chromosomes?

The information about chromosome length in base pairs (bp) is given in the column âSEQLENGTHâ:

chrom_length <- chromosomes[!duplicated(chromosomes$SEQNAME), c(3,2)] #SEQNAME & SEQLENGTH

chrom_length_other <- chrom_length[-which(chrom_length$SEQNAME %in% chrom), ]
scaffolds <- data.frame(SEQNAME = "S", SEQLENGTH = sum(chrom_length_other[grep("^KI[0-9]+|^GL[0-9]+", chrom_length_other$SEQNAME), "SEQLENGTH"]))

chrom_length <- chrom_length[which(chrom_length$SEQNAME %in% chrom), ] %>%
  rbind(scaffolds)


Now, I can combine the information about

  • how long each chromosome is,
  • how much of it is made up of genes and
  • how many genes there are it

into a single dataframe.

chromosomes_table <- left_join(chrom_length, chromosomes_num_table, by = c("SEQNAME" = "Var1")) %>%
  left_join(gene_bases, by = c("SEQNAME" = "Chromosome"))

colnames(chromosomes_table) <- c("Chromosome", "Chr_length", "No_genes", "Gene_lengths_sum")

Based on this information I am calculating

  • the number of genes per chromosome corrected for gene length and
  • the percentage of each chromosome that is taken up by gene sequences.
chromosomes_table$No_genes_by_Chr_length <- chromosomes_table$No_genes/chromosomes_table$Chr_length
chromosomes_table$Percent_of_Chr_covered_by_genes <- round(chromosomes_table$Gene_lengths_sum/chromosomes_table$Chr_length*100, digits = 2)
